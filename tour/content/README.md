# Welcome to the Vuego Tour

This interactive tour will teach you the fundamentals of Vuego, a Vue.js-inspired template engine for Go.

## What you'll learn

- **Variable Interpolation** - Output values using `{{ }}` syntax
- **Filters** - Transform values with pipes like `{{ name | upper }}`
- **Directives** - Control flow with `v-for`, `v-if`, and `v-show`
- **Components** - Build reusable templates with includes, slots, and layouts

## How to use

1. Read the lesson content on the left
2. Edit the template and data files on the right
3. Click **Render** to see your changes
4. Use **Previous** and **Next** to navigate between lessons

Click **Start** above or select a chapter below to begin!
